## 1. はじめに

### 1.1. 本規約について

この利用規約（以下「本規約」）は、proposal-generator（以下「本ツール」）の利用条件を定めるものです。本ツールを利用することにより、本規約に同意したものとみなされます。

### 1.2. 定義

| 用語 | 定義 |
|------|------|
| 本ツール | proposal-generator およびそれに付随するドキュメント、設定ファイル |
| 利用者 | 本ツールを利用する個人または法人 |
| 対象サービス | ランサーズ、クラウドワークスなど、本ツールがアクセスするクラウドソーシングサービス |

---

## 2. 利用条件

### 2.1. 利用目的

本ツールは、以下の目的での利用を想定しています：

- クラウドソーシングサービスへの提案文作成の効率化
- 個人または法人の業務における提案活動の支援

### 2.2. 利用資格

本ツールを利用するには、以下の条件を満たす必要があります：

1. 対象サービス（ランサーズ、クラウドワークス等）の利用規約に同意し、遵守していること
2. 本規約に同意していること
3. 日本国内法を遵守していること

### 2.3. 必要な権限・アカウント

- **Gemini API**: Google AI Studio のアカウントおよびAPIキー
- **GitHub API**: GitHubアカウント（オプション）
- **対象サービス**: 各サービスのアカウント（案件閲覧のため）

---

## 3. 禁止事項

### 3.1. 明示的な禁止事項

以下の行為を禁止します：

1. **過度なアクセス**
   - 短時間での大量リクエスト
   - サーバーに負荷をかける行為
   - DDoS攻撃に類する行為

2. **不正利用**
   - 対象サービスの利用規約に違反する行為
   - 虚偽の情報を含む提案文の生成
   - 他者になりすます行為

3. **悪意ある利用**
   - スパム提案の大量送信
   - 競合他社への妨害目的での利用
   - 詐欺・違法行為への利用

4. **知的財産権の侵害**
   - 著作権を侵害するコンテンツの生成
   - 商標権を侵害する行為

### 3.2. 技術的な制限事項

- 本ツールのリバースエンジニアリング
- 認証・認可機構の回避
- レート制限の意図的な回避

## 4. 免責事項

### 4.1. 保証の否認

本ツールは「現状有姿（AS IS）」で提供されます。以下について、明示または黙示を問わず、いかなる保証も行いません：

- **動作保証**: 特定の目的への適合性、中断のない動作
- **品質保証**: 生成される提案文の品質、採用率の向上
- **完全性**: 取得される情報の正確性、完全性
- **互換性**: 特定の環境での動作

### 4.2. 責任の制限

以下の事項について、開発者は一切の責任を負いません：

1. **本ツールの利用により生じた損害**
   - 案件の不採用、機会損失
   - データの損失、破損
   - 収益の減少、ビジネス上の損失

2. **外部サービスに起因する問題**
   - 対象サービスの仕様変更、サービス停止
   - Gemini API、GitHub API の仕様変更、エラー
   - ネットワーク障害

3. **利用者の行為に起因する問題**
   - 対象サービスの利用規約違反によるアカウント停止
   - 不適切な提案文の送信による評価低下
   - APIキーの漏洩

### 4.3. 利用者の責任

利用者は、以下について自己の責任において対応するものとします：

- 対象サービスの利用規約の確認・遵守
- 生成された提案文の内容確認・修正
- APIキーその他の認証情報の適切な管理
- 本ツールの利用により生じた結果への対応

## 6. プライバシー

### 6.1. 収集する情報

本ツールは、以下の情報をローカルで処理します：

| 情報 | 用途 | 保存場所 |
|------|------|---------|
| 案件情報 | 提案文生成 | メモリ上（保存しない） |
| GitHub情報 | 技術力アピール | メモリ上（保存しない） |
| 設定情報 | ツールの動作 | ローカルファイル |
| APIキー | 外部API認証 | 環境変数/ローカルファイル |

### 6.2. 外部送信

以下の情報が外部サービスに送信されます：

| 送信先 | 送信される情報 | 目的 |
|--------|--------------|------|
| Gemini API | 案件情報、プロンプト | 提案文生成 |
| GitHub API | ユーザー名 | 技術情報取得 |
| 対象サービス | HTTPリクエスト | 案件情報取得 |

### 6.3. データの保護

- APIキーはローカルに保存され、第三者に送信されません
- 案件情報はメモリ上でのみ処理され、永続化されません
- ログファイルには機密情報を含めないよう設計されています

## 7. 規約の変更

### 7.1. 変更の通知

本規約は、予告なく変更される場合があります。重要な変更がある場合は、以下の方法で通知します：

- GitHub リポジトリでの告知
- CHANGELOG への記載
- バージョンアップ時のメッセージ表示

### 7.2. 変更後の利用

規約変更後に本ツールを利用した場合、変更後の規約に同意したものとみなされます。変更後の規約に同意しない場合は、本ツールの利用を中止してください。
