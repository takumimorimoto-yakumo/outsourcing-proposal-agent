## 1. プロダクト概要

### 1.1. 目的

ランサーズ・クラウドワークスの案件情報を取得し、**案件内容に最適化された提案文**を自動生成するCLIツール。

### 1.2. 解決する課題

| 課題 | 解決方法 |
|------|----------|
| 案件ごとに提案文を書くのが面倒 | AI（Gemini）による自動生成 |
| 技術力をアピールしにくい | GitHubリポジトリ情報を自動取得・反映 |
| 提案文の品質にばらつきがある | テンプレート + AI生成で一定品質を担保 |

### 1.3. 対象ユーザー

- フリーランスエンジニア
- 開発チーム（クラウドソーシングで案件獲得）

```
│              ──▶   固定文読込    ──▶                            │
│                   (YAML)                                        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 4. 技術スタック

| カテゴリ | 技術 | 選定理由 |
|---------|------|----------|
| 言語 | Python 3.11+ | スクレイピングライブラリが豊富、人間らしい操作が可能 |
| スクレイピング | Playwright | 動的サイト対応、人間らしい振る舞いが可能 |
| AI/LLM | Gemini API | ユーザー指定 |
| 設定管理 | YAML (PyYAML) | 可読性が高く編集しやすい |
| CLI | Typer | モダンなCLI構築、型ヒント対応 |
| HTTP | httpx | 非同期対応、GitHub API用 |

**詳細**: [requirements/non-functional.md](./requirements/non-functional.md)

## 5. CLI使用方法

### 5.1. 基本コマンド

```bash
# ランサーズの案件から提案文を生成
proposal-gen https://www.lancers.jp/work/detail/XXXXXX

# クラウドワークスの案件から提案文を生成
proposal-gen https://crowdworks.jp/public/jobs/XXXXXX
```

### 5.2. オプション

```bash
# 出力先を指定（デフォルト: 標準出力）
proposal-gen <URL> --output proposal.txt

# クリップボードにコピー
proposal-gen <URL> --copy

# 詳細ログを表示
proposal-gen <URL> --verbose
```

### 5.3. 設定確認

```bash
# 現在の設定を表示
proposal-gen config show

# 設定ファイルを編集（エディタを起動）
proposal-gen config edit
```

## 6. 提案文の構成

生成される提案文は以下の6パートで構成される。

```
┌─────────────────────────────────────────┐
│ 1. 挨拶・自己紹介        [固定文]       │
├─────────────────────────────────────────┤
│ 2. 案件への理解・共感    [AI生成]       │
├─────────────────────────────────────────┤
│ 3. 提案内容・アプローチ  [AI生成]       │
├─────────────────────────────────────────┤
│ 4. 技術力・実績          [GitHub情報]   │
├─────────────────────────────────────────┤
│ 5. スケジュール・見積もり [AI生成]      │
├─────────────────────────────────────────┤
│ 6. 締めの挨拶            [固定文]       │
└─────────────────────────────────────────┘
```

**詳細**: [specs/generation.md](./specs/generation.md)
