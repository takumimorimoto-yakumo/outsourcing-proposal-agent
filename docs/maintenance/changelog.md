## [0.2.0] - 2025-12-19

### 追加
- **フロントエンドUI**
  - 案件一覧（テーブル/カード表示切替）
  - パイプラインサイドバー（案件 → 下書き → 提出済み → 進行中 → 履歴）
  - 設定画面（プロフィール設定、データ管理）
  - 優先度分析機能
- **プロフィール設定**
  - 基本情報（名前、自己紹介、リンク）
  - スキル（技術スキル、得意分野、自由記述）
  - 希望条件（カテゴリ選択、自由記述）
- **データキャッシュ**
  - ローカルストレージによる案件データキャッシュ
  - バックグラウンド更新（UIブロックなし）
  - 優先度スコアのキャッシュ

### 変更
- 設定画面をタブ構成に変更（プロフィール設定 / データ管理）
- プロフィールのサブタブ化（基本情報 / スキル / 希望条件）

### 削除
- プロフィールから不要項目を削除
  - 経験年数
  - 希望予算（最低/最高）
  - 稼働可能時間

### 修正
- テーブル表示エリアのオーバーフロー問題を修正
  - フレックスボックスによる高さ制御
  - `min-h-0` による縮小許可

## バージョン命名規則

```
MAJOR.MINOR.PATCH

例: 1.2.3
    │ │ └── パッチ: バグ修正（後方互換）
    │ └──── マイナー: 機能追加（後方互換）
    └────── メジャー: 破壊的変更
```

### 変更タイプ

- **追加 (Added)**: 新機能
- **変更 (Changed)**: 既存機能の変更
- **非推奨 (Deprecated)**: 将来削除予定の機能
- **削除 (Removed)**: 削除された機能
- **修正 (Fixed)**: バグ修正
- **セキュリティ (Security)**: セキュリティ関連の修正
